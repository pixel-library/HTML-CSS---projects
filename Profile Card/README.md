# 👤 Profile Card

A simple and responsive **Profile Card** built using **HTML** and **CSS**. This project displays user information in a clean and modern card layout with smooth styling and hover effects.

---

## ✨ Features

- Responsive Design
- Modern UI
- Profile Image
- Hover Effects
- Clean Layout
- Beginner Friendly

---

## 🛠️ Technologies Used

- HTML5
- CSS3

---

## 📂 Project Structure

```
Profile Card/
│
├── index.html
├── style.css
├── Images/
├── Screenshots/
│   ├── Profile-Card.png
│   └── Profile-Card.gif
└── README.md
```

---

## ⚙️ How It Works

- **HTML** creates the structure of the profile card.
- **CSS** styles the card with colors, spacing, animations, and responsive design.
